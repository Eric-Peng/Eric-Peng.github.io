我的静态博客搭建过程

  • hexo
hexo静态博客

欢迎各位来到我的github博客，希望此博客网站能给大家得到收获，做为第一篇的博客，首先给各位介绍我的博客破壳而出的全过程。

了解hexo

hexo是一款基于Node.js的静态博客框架。可以方便的生成静态网页托管在github中。据有关hexo粉指出，hexo速度快，指令简单，主题漂亮丰富吸引了他们，其实我是不会告诉你们吸引我的只是因为他是用nodejs开发的，别问为啥，任性。

博客环境搭建

1. 安装git与Node.js

下载并执行即可完成安装。

2. 安装hexo

利用 npm 命令即可安装

3. 创建项目文件夹初始化hexo环境

  1. 创建文件夹eric-peng.github.io

  2. 初始化hexo环境

  3. 安装依赖包 npm install

4. 本地运行

  1. 代码编译生成待发布的代码，代码默认生成在public文件夹下

  2. 运行nodeJs服务器查看本地效果，网址为localhost:4000

其中安装过程中可能会提示找不到该命令，原因是在Hexo 3.0 后server被单独出来了，需要安装server，安装的命令如下：npm install hexo-server –save 安装此server后再试，问题解决。

6. 添加主题

hexo下载默认会提供一种主题，若对主题不满意可修改主题。步骤如下，先去网站上下载对应的主题，此博客下载的是yilia主题。把下载的主题放置在于默认主题并列的位置，然后修改_config.yml。将配置项theme:landscape修改为theme:yilia.

7. 将本地代码部署到github

  1. 创建repository。名字对应eric-peng.github.io
  2. 编辑_config.yml。type类型改成git，repository改为clone Url 中对应ssl类型的url。

8. 主题配置

下载主题后，需要通过配置修改博客名，菜单，链接之类的，具体修改方法参照主题yilia对应的.READ.md

9. 添加评论

本博客系统采用多说评论平台。多说是一款追求极致体验的社会化评论框，可以用微博、QQ、人人、豆瓣等帐号登录并评论。界面漂亮，免登陆，且速度比disqus更快。是一款适用于国内的评论平台。yilia已内置了多说插件，故只需配置_config.yml即可。

安装完成后，请在后台设置你的域名shortname，多说对shortname的解释如下：当前站点注册的多说二级域名。例如：你注册了http://apitest.duoshuo.com/时，多说二级域名为apitest.其中的eric-peng是我多说设置的shortname值。
