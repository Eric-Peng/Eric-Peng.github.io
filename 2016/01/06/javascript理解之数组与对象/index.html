Object 是一个无序的集合，可以存放任意类型对象，所有其他对象都继承自这个对象。

对象只是一种特殊的数据。对象拥有属性和方法。 JavaScript 是面向对象的语言，但 JavaScript 不使用类。 JavaScript 基于 [prototype][1]，而不是基于类的。

JavaScript面向对象三大特征

封装：不考虑内部实现，只考虑功能使用
继承：从已有对象上，继承出新的对象
多态：所谓多态，就是指一个引用在不同情况下的多种状态，

数组部分

Array 对象：提供对创建任何数据类型的数组的支持。
